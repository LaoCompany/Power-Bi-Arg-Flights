# Análisis del Flujo Aéreo Argentino con Power BI

Este repositorio contiene un informe interactivo creado en Power BI que analiza el flujo aéreo en Argentina. El informe incluye datos sobre diversas aerolíneas y vuelos en la región.

## Contenido

- **Archivo Power BI:** [Nombre_del_Archivo.pbix](enlace_al_archivo.pbix) - Este archivo es un informe interactivo creado con Power BI. Puedes descargarlo y abrirlo localmente con Power BI Desktop o visualizarlo directamente en el servicio en línea de Power BI.

## Instrucciones de Uso

1. **Descarga del Archivo Power BI:**
   - Descarga el archivo [Nombre_del_Archivo.pbix](enlace_al_archivo.pbix) desde este repositorio.

2. **Apertura del Archivo en Power BI:**
   - Abre el archivo `.pbix` utilizando Power BI Desktop.

3. **Exploración del Informe:**
   - Explora las distintas visualizaciones y pestañas para analizar el flujo aéreo, las aerolíneas y otros aspectos relevantes.

4. **Actualización de Datos:**
   - Puedes actualizar los datos fácilmente en Power BI Desktop si tienes nuevas fuentes de información.
